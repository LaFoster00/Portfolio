WHITE ROOM - VR ESCAPE ROOM

VR, Tool and Gamedev

White Room is a room scale (4m x 4m) VR game for one player with an approximated playtime of 30 minutes. It was made using the Unity game engine and features mutliple different riddle mechanics centered around the direct physical interaction of the user with the environment.
